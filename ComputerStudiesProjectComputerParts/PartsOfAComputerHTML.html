Input Device

Input devices are devices that translate a phisical event into zero's and one's. Some examples are a keyboard, which takes button presses to convert to electronic signals, and a microphone, which takes vibrations in the air to convert.

Output Device

Output devices are devices that take electronic signals and convert them into phisical events. Some examples are a speaker, which takes electronic signals to convert to sound vibrations, and a monitor, which converts electronic signals into light.

Central Processing Unit (CPU)

A Central Processing Unit, or CPU, is what does the main processing of the computer. It is divided into two parts, the ALU, and the Control Unit.

Arithmetic Logic Unit (ALU)

An Arithmetic Logic Unit, or ALU, is what carries the bulk of the processing, made entirely for logic and arithmetic. It takes information from the Control Unit to calculate.

Control Unit

The Control Unit is the "leader" of the computer. It takes the imformation from the unput devices and gives the information to the ALU and sends those calculations to the output devices.

Random Access Memory

A Random Access Memory, or RAM, is the short-term memory a computer uses when processing. It requires energy to store information, but in return it is much faster than the long-term memory.

Optical Memory

Optical Memory is a kind of long-term memory used by reflecting light one direction or another, to represent zero's and ones. Some examples of such are CD's and DVD's.

Magnetic Memory

Magnetic Memory is a type of long term memory that stores information by magnets facing one direction or another, to represent zero's and one's. An example of such is a Floppy Disc.

Flash Memory

Flash Memory is a type of long-term storage that has a finite amound of uses, unlike the other types of memory. Some examples are SSDs and SD cards.

Logic Gates

Logic Gates are electronic components that takes two electronic inputs and provides an output. Some examples of such are an AND gate and an OR gate.
